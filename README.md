# Asteroid V2
> 바벨을 쓰는 에스터로이드 V2

## Requirement
- Node.js (10.x.x 추천)
- MariaDB
- Yarn
- Window x64 (10 추천)

## Installation
- `yarn` 실행
- src/lib/config/config.inc.json 을 config.json 으로 이름 변경
- config.json 수정
- `yarn build` 실행
- `yarn start` 실행
